# GDc — Google Colab Projects: ML Crash Course

A collection of Google Colab notebooks covering projects and exercises from the **Machine Learning Crash Course (MLCC)**. Each notebook is self-contained and can be run directly in Google Colab without any local setup.

---

## 📚 About

This repository contains hands-on project notebooks that accompany the [Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course) by Google Developers. The notebooks explore core ML concepts through practical exercises, including:

- Data exploration and preprocessing
- Linear regression and logistic regression
- Neural networks
- Feature engineering
- Model evaluation and validation

---

## 🚀 Getting Started

### Run in Google Colab

1. Browse the notebooks in this repository.
2. Click on any `.ipynb` file to open it.
3. Click the **"Open in Colab"** badge or button to launch it directly in Google Colab.
4. Run each cell sequentially using **Shift + Enter** or the ▶ button.

> **Tip:** Sign in with a Google account to save your own copy to Google Drive via **File → Save a copy in Drive**.

### Run Locally

If you prefer running notebooks locally:

```bash
# 1. Clone the repository
git clone https://github.com/kenechukwuosele/GDc.git
cd GDc

# 2. Install Jupyter and required dependencies
pip install jupyter numpy pandas matplotlib scikit-learn tensorflow

# 3. Launch Jupyter Notebook
jupyter notebook
```

## 🛠️ Prerequisites

| Tool | Purpose |
|------|---------|
| [Google Account](https://accounts.google.com/) | Required to run notebooks in Google Colab |
| Python 3.7+ | Required for local execution |
| TensorFlow / Keras | ML framework used in the projects |
| NumPy, Pandas | Data manipulation |
| Matplotlib | Data visualization |

All dependencies are pre-installed in Google Colab, so no local setup is needed when running there.
