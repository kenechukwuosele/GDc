# GDc

Colab notebooks for projects and exercises from the Google Machine Learning Crash Course (MLCC).

## What's inside

- `ml/cc/exercises/binary_classification_rice.ipynb` — build and evaluate a binary classifier using a Turkish rice dataset.

## Quick start

Open the notebook in Colab and run the cells top-to-bottom. All required Python packages are installed in the notebook.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/kenechukwuosele/GDc/blob/main/ml/cc/exercises/binary_classification_rice.ipynb)
